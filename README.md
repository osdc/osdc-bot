This bot lives at [https://gitter.im/osdc/Hackers](https://gitter.im/osdc/Hackers)

### Installation:

```
$ make install
```

### Running locally with I/O to stdin and stdout:

```
$ make run
```

### Running in production

```
$ make serve
```

### Testing:

```
$ make test
```
